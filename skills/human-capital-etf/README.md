# Human Capital ETF Skill

> Map, diagnose, and rebalance non-financial human capital through **Core, Distribution, Meta, and Growth**.

Human Capital ETF is an interactive Agent Skill and an open framework by **Qiaomai Liu**. It helps a person identify what work and learning have actually accumulated, distinguish portable from context-specific capability, find conversion bottlenecks, and create a focused 30- or 90-day rebalancing plan.

It is **not** a security, financial ETF, personality test, universal score, or substitute for medical, psychological, legal, or financial professionals.

## The four roles

| Role | Function |
|---|---|
| **Core** | Protect validated capability, health, energy, habits, judgment, and operating capacity |
| **Distribution** | Externalize and verify capability through proof, communication, relationships, channels, information, and opportunity access |
| **Meta** | Observe, learn, select, stop, update, document, automate, and rebalance |
| **Growth** | Build and test new capability through education, training, practice, experiments, and real projects |

The ETF metaphor contributes **role clarity, diversification, drift detection, lifecycle movement, and rebalancing**. It does not imply a market price for a person or fixed allocation percentages.

## Use it

Explicit invocation:

```text
Use human-capital-etf to map what I accumulated in five years of work and create a 90-day rebalance.
```

Codex-style invocation:

```text
$human-capital-etf I keep learning AI tools but have no durable proof of work. Diagnose the bottleneck.
```

Natural-language triggers:

```text
I have worked for years but cannot explain which abilities would survive outside my company.
```

```text
I am doing too many learning projects. Help me decide what to stop.
```

```text
I have strong skills, but nobody outside my team can see or verify them.
```

## Interaction modes

- **Guided diagnosis** — one question at a time;
- **Context dump** — paste a resume, biography, project list, review, or notes;
- **Best-guess scan** — a provisional map from limited context;
- **Rebalance review** — compare intended and actual allocation over time.

## Becker integration

Gary S. Becker's human-capital economics provides an investment lens for education, on-the-job training, health, information, mobility, direct and opportunity costs, returns, depreciation, and general versus specific training.

Human Capital ETF adds a portfolio-management layer. Becker did not propose Core, Distribution, Meta, and Growth. The framework uses two simultaneous descriptions:

1. what the holding consists of;
2. what portfolio role it currently serves.

A Growth holding can become Core after repeated use and validation. Core and Growth gain external evidence through Distribution. Meta interprets feedback and changes allocation.

## Install

Copy this directory into an Agent Skills-compatible location.

Codex user scope:

```text
$HOME/.agents/skills/human-capital-etf/
```

Codex repository scope:

```text
$REPO_ROOT/.agents/skills/human-capital-etf/
```

The GitHub repository distributes the instructions; an Agent must load `SKILL.md` to execute them.
